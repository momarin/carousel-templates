# Owl Carousel de frutas favoritas, utilizando Bootstrap 5 e contendo:
    - responsividade sm, md e lg, de acordo com bkpoints do Bootstrap 5;
    - imagens redondas feitas no custom css (alteração do design original para treino de flexbox);
    - animação @keyframes;
    - script do owl carousel:
        - definição da responsividade de cada screen;
        - remoção dos botões de navegação nav;
        - definição de loop automatico;
        - estilização total dos dots do owl carousel;
        - ativação de autoplay com definição de tempo;
        - smartspeed;
# Observações:
    -> Sobre a classe offset do Bootstrap:
        - é um recurso que permite criar margens ou espaçamentos entre colunas em um layout de grade (grid system). Ele empurra a coluna para a direita, criando uma margem à esquerda, sendo útil para centralizar elementos ou criar layouts assimétricos
    -> a classe img-fluid na tag img pode afetar o custom css .img-ara img.
    
    !IMPORTANT
    o Owl Carousel exige o script minificado do JQuery, além do seu próprio, e o seu CSS minificado;
